# Security Policy

## Supported Versions

Only the latest release is actively maintained.

| Version | Supported |
|---------|-----------|
| Latest  | ✅        |
| Older   | ❌        |

## Reporting a Vulnerability

**Please do not open a public issue for security vulnerabilities.**

Report them privately using [GitHub's Security Advisory feature](../../security/advisories/new). Include:

- A description of the vulnerability and its potential impact.
- Steps to reproduce (if applicable).
- Any suggested fix you may have.

You can expect an acknowledgement within a few days and a resolution or status update within 14 days.
